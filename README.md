#### Libraries

```
$ react-navigation
$ axios
$ lottie-react-native
$ react-native-fast-image
$ react-native-gesture-handler
$ react-native-modalize
$ react-native-reanimated
$ react-native-svg
$ react-native-ui-lib
$ realm

```
#### Storage
```
$ Realm local database
```
#### Get Started 
```
$ git clone https://github.com/cnpsn/ECommerceExample.git
$ cd ECommerceExample
$ npm install
$ cd ios/
$ pod install
$ cd ..
$ npx react-native run-ios
```
